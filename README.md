# GRAPPA
vim-like paint application for mac(OS X), Linux(CentOS) using GLUT

Extract the zip file and change directory to 'GRAPPA-master', then make 'GRAPPA'
```
$ cd ~/Downloads/GRAPPA-master
$ make
$ ./bin/GRAPPA
```
